```python
class Solution:
    def minPairSum(self, nums: List[int]) -> int:
        start = 0
        end = len(nums) - 1
        result = 0
        nums.sort()
        while start < end :
            result = max(result, nums[start] + nums[end])
            start += 1
            end -= 1
        return result
```
